# Nutri-Sport

## Setup Instructions
1. Clone the repository:
   ```
   git clone https://github.com/ewen-69/Nutri-Sport.git
   cd Nutri-Sport
   ```
2. Install the required dependencies:
   ```
   npm install
   ```

## Features
- User-friendly interface for tracking nutrition
- Provides personalized meal recommendations
- Integrates with fitness applications

## Usage Guide
1. Start the application:
   ```
   npm start
   ```
2. Follow the on-screen instructions to log your meals and activities.
3. Use the dashboard to view your nutrition statistics and receive meal suggestions based on your goals.
